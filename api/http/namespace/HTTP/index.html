Constructor

Headers

  • Headers(values: Iterable<readonly [string, string]> | Record<string, string>): Headers

  • HTTP header record constructor

    @example
    const headersFromIterable = Headers([
     ['key1', 'value1'],
     ['key2', 'value2']
    ]);// { key1: 'value1, key2: 'value2' }
    const headersFromObject = Headers({
     key1: 'value1',
     key2: 'value2'
    });// { key1: 'value1, key2: 'value2' }

request

  • request<Value>(requestObject: Request<Value>): Task<Value, HTTPError>

  • Return a new @w5s/core!Task that will send an HTTP request

    @example
    const responseTask = HTTP.request({
      url: 'http://someurl.com',
      parse: parseText,
    });
