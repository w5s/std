Callable

Callable

  • Callable<T>(properties: T): CallableFunction<T>

  • Return a new function from callable interface

    @example
    const myCallable = Callable({
      [Callable.symbol]: (arg: number) => arg,
      myMethod: () => {},
    });
    // myCallable(1)
    // myCallable.myMethod();

    Type parameters

    • T: Callable<AnyFunction>
